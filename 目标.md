# MongoDB快速上手
MongoDB用起来-快速上手&集群和安全系列

课程目标
    - 理解MongoDB的业务场景、熟悉MongoDB的简介、特点和体系结构、数据类型等。
    - 能够在Windows和Linux下安装和启动MongoDB、图形化管理界面Compass的安装和使用
    - 掌握MongoDB的基本常用命令行实现数剧的CRUD
    - 掌握MongoDB的索引类型、索引管理、执行计划。
    - 是哟个Spring Data MongoDB完成评论业务的开发
